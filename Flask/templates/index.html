HOME FIELD ADVANTAGE

Is it a thing?

Project Overview

Home-field advantage is a long-running idea in sports. The home team knows the turf, they’ve got more fans in the stands, and they got to sleep in their own beds rather than traveling possibly thousands of miles and staying in a hotel. But is home-field advantage really all it’s cracked up to be?

The purpose of this project was to gather data on the NBA, NFL, and the MLB and determine whether teams won more games at home or on the road and if it was significant or not.




Data

NBA - nba_api.stats

NFL - Kaggle Dataset

MLB - Kaggle Dataset




Methods

NBA - Using the API listed above, a json file was pulled for NBA game stats from 2012 to 2017. That data was then cleaned and formatted in a way that is easy to replicate, then sent to MongoDB.

NFL - The Kagle dataset provied game stats for all teams from 2012 to 2016. This data was then cleaned and formatted in the same way the NBA data was and sent to MongoDB.

MLB - This Kaggle dataset orovided game information dating back to 1871 but only seasons 2012 to 2017 were used. This was then put into the same format as the others and sent to MongoDB.
